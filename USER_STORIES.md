# User Stories

## New

### Story 1

**As a** shopper  
**I want** to browse gifts by category  
**So that** I can find appropriate gifts faster  
_Label:_ `new`

### Story 2

**As a** shopper  
**I want** to search for gifts by keyword  
**So that** I can find specific items I have in mind  
_Label:_ `new`

## Backlog

### Story 3

**As a** registered user  
**I want** to save favorite gifts to a wishlist  
**So that** I can purchase them later  
_Label:_ `backlog`

### Story 4

**As a** user  
**I want** to view recent reviews for a gift  
**So that** I can make informed purchasing decisions  
_Label:_ `backlog`

## Technical Debt

### Story 5

**As a** developer  
**I want** to refactor the authentication middleware  
**So that** it is easier to maintain and test  
_Label:_ `technical debt`

### Story 6

**As a** developer  
**I want** to update the database schema for user profiles  
**So that** we can store additional preference data cleanly  
_Label:_ `technical debt`

## Icebox

### Story 7

**As a** administrator  
**I want** a dashboard to view daily sales metrics  
**So that** I can track business performance  
_Label:_ `icebox`

### Story 8

**As a** shopper  
**I want** to share gift lists on social media  
**So that** friends can see what I like  
_Label:_ `icebox`
